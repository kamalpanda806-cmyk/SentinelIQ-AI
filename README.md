# 🛡️ SentinelIQ AI

## AI-Powered Security Operations Center (SOC) Incident Triage Assistant

### 🚀 Project Overview

SentinelIQ AI is an AI-powered cybersecurity assistant that helps Security Operations Center (SOC) analysts investigate security incidents faster and more accurately.

It analyzes security logs, predicts threat severity, maps MITRE ATT&CK techniques, identifies root causes, and generates professional incident reports.

---

## 📌 Problem Statement

Organizations receive thousands of security alerts every day.

SOC analysts spend significant time identifying which alerts are genuine threats.

SentinelIQ AI reduces investigation time using AI-powered reasoning.
